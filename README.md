# Java Mini Projects

This repository contains Java programs and mini projects created to practice core Java concepts and problem-solving skills.

## Topics Covered
- Java basics  
- Object-Oriented Programming  
- Collections Framework  
- Exception handling  
- File handling  
- Multithreading  
- String and number manipulation  

## Mini Projects
- Banking System  
- Banking System with Transaction History  
- Student Management System  
- Employee Management System  
- Tic Tac Toe  
- Notes Application  
- Password Strength Checker  
- Email Validator  
- Number Converter  

## How to Run
Compile and run any `.java` file using a Java IDE or command line.
